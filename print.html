Cache + Pipeline

cache/ + pipeline/ — nunca classificar o mesmo PR duas vezes

pythondef make_cache_key(*args: str) -> str:
    return hashlib.sha256("\x00".join(args).encode()).hexdigest()

def compose(*fns: Callable) -> Callable:
    return reduce(lambda f, g: lambda x: g(f(x)), fns, lambda x: x)
  • SQLite WAL · thread-safe · sobrevive ao Docker rebuild
